Problem Solving and Programming in Python

Overview

Programming capabilities in GIS software allow users to process data and automate repetitive tasks. Often a few lines of code can save time or can accomplish tasks that would otherwise not be feasible. In this course students will develop their core problem solving and algorithm design skills and implement solutions in a current programming language. This course focusses on programming fundamentals that are applicable to many programming languages used in the Geospatial industry. This course also serves as the foundation to the other programming and technical courses in the GIS Specialist programs.

A good algorithm should solve a problem:

CORRECTLY,EFFICIENTLY, READABLY, And In a user-friendly fashion

Problem solving process:

  • Requirements - formalization of idea
  • Specification - a detailed user’s view of the problem
  • Algorithm - a programmer’s view of the problem (using structured logic)
  • Test Algorithm (Desk Check) – step through the algorithm using actual data and determine if it gives desired results
  • Implementation – coding of the algorithm in a programming language
  • Test Application – to find errors and alter code to correct them (debug)
  • Documentation – throughout the entire process
  • Maintain and upgrade – as necessary
